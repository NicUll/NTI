# Responsiv design

## Övergripande arbetsgång

1. Välj innehåll
2. Bestäm tema
  * Typografi
  * Färger
  * Former
  * Avstånd
3. Skissa design
  * Skärmstorlekar
    * Mobil
    * PC/stor skärm
    * (Tablet)
  * Tydlig meny
4. Strukturera arbetet
  * Dela in i byggblock
  * Gruppera element
  * Markera behållare
5. Skriv HTML skelett
  * Skriv utan innehållet först
  * Börja utifrån och in (behållare först, detaljerna sen)
  * Fyll i innehållet
6. Kontrollera
  * Kolla på sidan utan CSS, ser den ok ut?
  * Validera HTML
  (kolla av med mig)
