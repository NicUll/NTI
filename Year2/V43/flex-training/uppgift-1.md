# Uppgift 1

## Gör ett skelett

1. Skapa filer:
  * index.html
  * style.css
  
2. Skriv grundläggande html:
  * Doctype
  * html - inklusive *lang*
  * head
  * body
  
3. Fyll i header:
  * <meta> med charset
  * <title> valfritt
  * <link rel="stylesheet" href="style.css">
  
4. Gör ett skelett i body
  * <header>
  * <main>
  * <footer>
